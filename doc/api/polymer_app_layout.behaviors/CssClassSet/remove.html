bool remove(
Object value
)

Remove the class value from element, and return true on successful removal.

remove and removeAll are the Dart equivalent of jQuery's removeClass.

value must be a valid 'token' representing a single class, i.e. a non-empty string containing no whitespace. To remove multiple classes, use removeAll.

Source

/**
 * Remove the class [value] from element, and return true on successful
 * removal.
 *
 * [remove] and [removeAll] are the Dart equivalent of jQuery's
 * [removeClass](http://api.jquery.com/removeClass/).
 *
 * [value] must be a valid 'token' representing a single class, i.e. a
 * non-empty string containing no whitespace.  To remove multiple classes, use
 * [removeAll].
 */
